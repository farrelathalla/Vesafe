# Product Sense

VeSafe exists to shorten the path from spatial risk to corrective action.

- Every major workflow should reduce time-to-triage for K3 officers and safety auditors.
- Labels must be in plain Indonesian K3 language, referencing specific locations and risks.
- Recommendations must be specific, actionable, and reference Indonesian standards (CPOB, Permenaker, SMK3, SNI, ISO 45001).
- A user should understand what to inspect in the facility without parsing internal model jargon.
- Confidence and evidence matter because the system works with partially observed public imagery.
