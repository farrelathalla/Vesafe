# Facility Onboarding

The onboarding flow should:

1. Let a safety officer register a facility (name + address).
2. Allow manual upload of facility interior photos (JPG, PNG, WebP; up to 20 per session).
3. Trigger world model generation from the uploaded photos.
4. Support additional supplemental uploads at any time to improve coverage.

All imagery is provided manually by the safety officer — there is no automatic acquisition from external sources. Photos should follow the capture guide in `PHOTO_CAPTURE_GUIDE.md` to ensure sufficient overlap for Gaussian Splatting reconstruction.
