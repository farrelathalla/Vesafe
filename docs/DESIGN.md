# Design

The UI should feel operational, industrial-safety-focused, and decisive rather than generic SaaS.

- Use warm neutral backgrounds with strong K3-alert accents (red for CRITICAL, amber for HIGH, teal for flow issues).
- Keep the world model viewer dominant on the page.
- Make domain and severity meaning visible without needing a legend hunt.
- Favor dense, legible information panels over decorative chrome.
- Preserve route-level clarity: dashboard, coverage, model, and report each need a single obvious job.
- Domain labels should use Indonesian K3 terminology (Pencegahan Kontaminasi, Tanggap Darurat, etc.).
